# IE Policy - Lab 4

## 1. DATE
- **Визначення**: Календарна дата у форматі ISO або українському цифровому.
- **Нормалізація**: YYYY-MM-DD.
- **Негативні приклади**: Версії ПЗ (2.0.1).

## 2. GEO_ID
- **Визначення**: Цифровий ідентифікатор об'єкта в базі GeoNames.
- **Нормалізація**: Числовий рядок.
- **Edge Case**: Номери телефонів або поштові індекси (фільтруємо за довжиною та контекстом 'ID').

## 3. LOCATION
- **Визначення**: Назва населеного пункту, що йде після маркерів (місто, обл. тощо).
- **Нормалізація**: Capitalized string.
- **Негативні приклади**: Прізвища після слова 'вул.' (якщо це не назва міста).
